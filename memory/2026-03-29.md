### 2. Key User Decisions

**V31 Marius TPL:**
- Symbol Grid: All 32 symbols with color coding (Green=Buy, Red=Sell, Grey=Neutral)
- Score format: "7/10" (conditions met/total)
- Click symbol → Modal with details (RSI, Stoch, EMA) + Open MT5 button
- Thinking Process: Real-time terminal showing each analysis step
- Incomplete Setups: List with Force Trade, Modify, Delete actions

**V32 London Breakout:**
- London Time (GMT) + adaptive Session Timer
- OR High/Low (08:00-08:15 formation)
- Asia Range for profit targets
- Type A (strong breakout) and Type B (pullback) setups
- Popup + sound notification on breakout

**V33 NY Breakout:**
- Pre-Session High/Low (11:30-13:00 EST)
- EURUSD focus

**Live Sync:**
- WebSocket for critical data (decisions, executions)
- Polling 1s for general status
- Sub-1-second updates required
- Terminal live showing robot thinking process

**Queue & Journal:**
- Queue: Open Position commands with Execute/Cancel/Modify
- Journal: Real-time terminal, filterable by robot/level/period
- Export to CSV

**UI/UX:**
- LED animated pulse when running (green)
- Tabs: Overview, Analysis, Trades, Logs, Settings
- Accordion sections inside tabs
- Error handling with auto-retry
